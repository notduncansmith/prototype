Hello World!

And goodbye “environment setup”

Tired of logging hours and hours of time to "environment setup"? Well, I was too. Getting started on a new project required downloading Boostrap, jQuery, touch index.html, etc. Nasty. But, no more!

With this nifty prototype application, you can hit the ground running on your next project. Just clone this repo, run install.sh (just gathers dependencies and does the initial build), and you're good to go!

What's In The Box

jQuery

Because Bootstrap needs it, and let's face it: you'll probably want it.

Bootstrap

The most widely-used grid framework in the known universe.

Gulp

Because you should only need to run one command to get your dev environment running, and it's a lot less hairy than Grunt.

LiveReload

Do you like constantly hitting F5? Me neither.

LESS

Because LESS is powered by Javascript, and plays nicely with Bootstrap.

Bower

Because managing front-end dependencies by hand is for cavemen.

.gitignore

You'll probably want to use Git, and you probably won't want your dependencies tracked.

Jekyll architecture

Templating for maximum DRYness, and deploys to GH Pages with ease.

Flat UI

Prototype comes bundled with the Flatly theme, giving your site a consistent and beautiful look and feel.
